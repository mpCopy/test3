Setting Layer Properties

After the PCB Importer parses the ODB (or IPC2581) data the user should check to see that the correct layer thickness has been imported. Unfortunately, ODB++ does not make it easy to pass this information and it is generally missing from the ODB++ database. Hence the user will have to enter the thicknesses using the Properties dialog.

To access the dialog that sets the thickness click on the Properties ... button that is located below the column labeled Details.


The Layer Properties Dialog will Open


The values shown for Thickness are default ones and do not represent the board thickness. To modify the values:

  • first highlights the layer (or layers) by clicking on the stackup number (holding the ctrl key down enables one to select multiple layers)

  • Enter a Thickness and a Material, color and Conductivity, Permittivity and Permeability values.

  • Click on Modify to update the selected layers. Then click OK to make your changes permanent.

Notes on Thickness, Material and Electrical Properties

The layer data passed to ADS concerning thickness, material, and electrical properties is used to create a substrate defnition. One can always edit the substrate parameters in ADS to change the thickness or other properties. Since the ADFI file is two dimensional data and the 3D extrusion is done later, no geometric information is lost or needs to be modified by changing layer thickness.

Adding or Modifying to Available List of Materials

The pulldown that allows a user to select a material reads the available materials and their electrical properties from a text file called ntxgmatlib.txt which is located in the NETEX-ADS installation directory.

A user can edit this file to add a material and its electrical properties.
