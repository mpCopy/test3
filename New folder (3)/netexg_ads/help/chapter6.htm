Extractions

While it is possible to import an entire PCB layout into ADS this is not always practical since many high speed boards are so large that one cannot actually do any useful simulation on the entire board. Instead, users generally focus on either some critical nets or a small critical region.

The ADS Layout editor was not designed to handle extremely large databases so manually deleting all the data you don't need to simulate is very time consuming.

The PCB Importer allows the user to specify:

    a) one or more nets and any conductors in proximity to those nets

    or

    b) one or more windows of the layout.


Selection by Net

If you know the net or nets you wish to select then you can specify the names and the PCB Importer will extract all conductors associated with the names.

To do this, click on the Extract button from the main menu and the Exract Dialog will open as seen below:


There are several ways to select the nets.

    You can click on a net name seen in the Available Net Names field. Use the ctrl and shift keys to select multiple nets as you would in the file manager.

    You could type in the name of the net directly into the Nets to Extract field.

    You could type in a DOS wildcard expression into the Nets to Extract field. Any nets matching the expression will be extracted.1



[1] if you use a wildcard expression such as A* you will not be able to see which matching nets are selected. So make sure that you pick an expression that collects only the nets you want and does not inadvertently select other nets.



Proximity Nets

Extracting just the nets to simulate is of little value if you can't take into account all of the other conductors in close proximity to the selected nets that have an influence through capacitive (and possibly inductive) coupling.

Hence, if you select extraction by net, the user can also specify a "proximity distance." The part of any conductor that falls within this distance is also extracted.

Extractions by Window

For some types of layout it may be better to extract a small window from the layout. In this case one gets everything inside of the window. (Components that cross the window are extracted in their entirety.)

To extract by window, open the Net and Window Extractions dialog box and then click on the Define ... button. This will launch the GBRVU program and display all the conductor and drill data in GBRVU's window.


Here is the extraction toolbar and the functions of the buttons


Rectangular Selection

To make a rectangular selection make sure the rectangular selection button is pressed in, and the mode is set to Single. then use the cursor to define the corners of a window. (Click and hold, drag, release) If you are satisfied with your selection click the Finish button.


If you are not satisfied, you can merely make a new selection.

If you want to create an irregular shape you can do it in rectangular mode by using the + and - toolbar buttons to add to the current selection or remove from the current selection.


Polygonal Selection

Press the polygonal selection toolbar button. Then move the cursor to the first vertex and click up/down. Move to the next vertex and click again. Continue until you have created the desired polygon shape. Clicking on the original starting point will complete the selection.



Panning and Zooming

Often it is necessary to pan and zoom before making the selection. One way to do this is to uncheck both the rectangular and polygonal selection buttons. Once those are unselected the regular pan and zoom buttons on the gbrvu toolbar become active.

When using the zoom window toolbutton, Dragging from upper left to lower right zooms in. Dragging from lower right to upper left zooms out.



Closing the GBRVU Display

You must close the GBRVU display either by clicking on the Finish button, or by closing the entire GBRVU program. You will not be able to continue using the PCB Importer dialog until GBRVU is closed (not minimized).
