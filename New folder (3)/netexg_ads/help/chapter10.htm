Miscellaneous Issues and Trouble Shooting

This section covers a variety of miscellaneous issues and trouble shooting that don't come under the other topics.


Q. I have a ODB++ file in zip format but it doesn't open when I ask the PCB Importer to import it. What should I do?

A. Unless the ODB++ file has been assembled in .tgz or .tar format, it must be unzipped first. Then select the directory option when opening it and point to the directory that contains the following directories:

[fonts]
[matrix]         required
[misc]
[steps]          required

Q. I am not seeing my components, ports or any nets when I get the data into ADS. Why not?

A. There are a couple of possible reasons:

    i) the ODB (or IPC) file did not contain component and net information. While this is rare, it is possible to produce a valid ODB file that does not have component and net information.

    ii) you did not load the layer COMP_PLUS_TOP and/or COMP_PLUS_BOTTOM. These layers contain the component placement and footprint information.


Q. I selected a couple of nets and a proximity distance but I am getting the entire board in ADS. Why?

A. You may have selected a net such as GND, VSS, VCC that essentially covers the entire board. When that net is expanded it will grab everything around it.

or

You may have accidentally selected a proximity distance that is quite large. For example if you thought the data was in mm and you selected a 2.5 mm expansion distance but the data was actually in inches then you would potentially get the entire board.


Q. Why, if the input is ODB++, do you provide a Gerber viewer?

A. The PCB Importer does not view ODB++ natively. Intstead, it converts the geometry layers into Gerber and we use an existing viewer, GBRVU to show the layout.


Q. Does the ODB++ and the IPC2581 includes all the substrate parameters?

A. ODB++ does not. You get the stackup order but no information about the thickness of each layer, the material or electrical properties. Therefore this information should be entered using the Properties ... button from the main menu.

IPC 2581 has a structure for including the thickness and electrical properties. However there is no guarantee that the PCB software that creates the IPC2581 file will actually populate the file with the correct information.


Q. Can I model the effects of the solder mask on the top and bottom of the PCB?

A. Currently you cannot as this data is not imported and transferred to ADS by the PCB Importer.
