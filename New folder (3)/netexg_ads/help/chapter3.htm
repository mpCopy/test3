Importing Files

The first step in importing PCB manufacturing data is to open the file (or the directory hierarchy). This is done from the File pull down available in the main dialog box:

ODB++

To import an ODB++ file use the File | Import ODB++ menu item.


The import dialog will open. You must choose between opening a file (in .tgz format) or a directory hierarchy by pressing the correct radio buttom selector.


In the screen shot above we have selected Directory so we use the Open button and browse to the directory that is the "top" of the ODB++ hierarchy.


Once the directory is selected and the user hits OK, the program will do the following:

    a) make a copy of the ODB++ hierarchy in a temporary location.

    b) scan the ODB data and populate the list of STEPs and Layers


Selecting The Step and Layers

Most PCB Design software will only create a single STEP. (In principle, the ODB++ file could have multiple circuits - each represented by a STEP along with an array; again, another STEP - but this is only seen when preparing a circuit for panelization.)

The only layer types that the user should import are:

component

signal

power gnd

drill

These are selected by holding the Ctrl key down and clicking on each layer to import.

If you select layers of type other than the supported types no harm is done. However if you fail to select a layer of the required type important sections of the layout data will not be transfered to ADS.


Note that there may be components on both top and bottom of the board and one should select both top and bottom components. There are often multiple drill layers to account for buried and blind vias so select all layers of type drill.

When you are satisfied that you have selected the proper layers click on the Import button.


Previewing Data

If you wish to preview one or more layers before doing your import, highlight the layer or layer(s) by clicking on it and then click on Preview. This will launch the GBRVU program and show you the data in a new window.
