ADFI Output

The PCB Importer produces an output file known as ADFI. This is a text based file which includes substrate definition, conductor geometry, vias and component/net information.

The user only needs to set the location and name of the output file using the output dialog.

The dialog is accessed from the Output button in the main menu.



The output directory defaults to the one passed to it by ADS. However a user can change this by using the browse button and navigating to any desired directory.
