Previewing Layers

After you have selected your ODB or IPC2581 data, the PCB Importer scans the file and presents the user with a list of steps and layers.


Should you want to view one or more of the layers, first higlight them and then click on the Preview button.


The PCB Importer will convert the selected layers into an "intermediate" Gerber file and then launch GBRVU - passing it the list of layers to display.


If you need to turn off some layers use the Layers dialog.



Notes

    Empty Layers - it is quite common to find ODB++ and IPC2581 layers listed in the Import dialog, that when viewed are empty. Very likely this is because the PCB layout tool creates an empty layer but the user either does not configure any output for that layer or does not draw anything in that layer.
