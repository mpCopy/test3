Introduction to GBRVU

GBRVU is a layout viewer included with the PCB Importer. It allows users to preview data prior to importing into ADS and also to select windows of the the layout for extraction.

While the PCB Importer is intended to convert ODB++ and IPC2581 data files into ADFI, it cannot display these formats natively. Instead, it converts each layer into an intermediate Gerber file which it then displays using GBRVU.


In addition to pan and zoom, the viewer allows a user to identify an element and to make simple distance measurements.
