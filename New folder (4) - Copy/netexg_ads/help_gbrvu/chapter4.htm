Extracting Windows

GBRVU can be used to extract a small window (or multiple windows) from a large layout. This reduces the data passed to ADS and the time ADS will need to import and simulate the layout.

In order to extract a window, GBRVU must be launched from the Net Extractions and Windowing dialog button using the Define button.


Launching GBRVU from this window loads all the layers selected for simulation and also adds a new toolbar to GBRVU.


Rectangular Selection

To make a rectangular selection make sure the rectangular selection button is pressed in, and the mode is set to Single. then use the cursor to define the corners of a window. (Click and hold, drag, release) If you are satisfied with your selection click the Finish button.


If you are not satisfied, you can merely make a new selection.

If you want to create an irregular shape you can do it in rectangular mode by using the + and - toolbar buttons to add to the current selection or remove from the current selection.


Polygonal Selection

Press the polygonal selection toolbar button. Then move the cursor to the first vertex and click up/down. Move to the next vertex and click again. Continue until you have created the desired polygon shape. Clicking on the same point twice will complete the selection.

To remove an undesired polygonal selection you can use the Minus (-) Toolbar button.


Once you are satisfied with your selection you can click on the SELECT button. This will pass your selection window to the PCB Importer and close GBRVU.
