Viewing and Panning Controls

When GBRVU is launched from the PCB Importer, one or more layers will be displayed. The user can pan and zoom as well as turn layers on and off or assign different layer colors.


Panning and Zooming

There are several different ways to pan and zoom.

    Pan - hold the middle mouse button (or wheel) down and drag the cursor. The display will lock to the cursor until you let go of the middle button.

    Pan - use the left-right-up-down arrow buttons on the keyboard. Each click of the key will move the center of the display approximately 1/4 of the width or height.

    Zoom - select the magnifying glass cursor. Dragging it from upper left to lower right will zoom in. Dragging it from lower right to upper left will zoom out.

    Zoom - use the middle mouse wheel. Roll Away zooms in. Roll inward zooms out.

    Zoom - use keyboard Ctrl + to zoom in and Ctrl - to zoom out (same as Adobe Photoshop)



Layer Control

If you have selected more than one layer to preview, or if you are doing a window extraction and need to turn off layers so that you can isolate a piece of circuitry you will want to use the layer dialog to control which layers are displayed.

Access this dialog using the File | Layer ... pulldown menu item.


You should see a layer dialog that looks like this:


Actions

    Layer On/Off - Check or uncheck next to the layer to turn display on/off. You must click either Apply or Close for the display to update.

    Paint/Scratch/Negative - Do not change the Paint/Scratch/Negative Button as this information is derived from the ODB++ data.

    Color - click on the rectangle to get a color dialog. Select from one of the available colors.


Re-Odering the Display Painting

It may be useful occasionally to re-order the display. GBRVU paints the display in the order shown. It is possible that data you would like to see is covered by a layer that is painted more recently.

If turning off the covering layer is not acceptable, you could click on Reorder ... and use that dialog to move a layer up or down in the display order.

To re-order, first select a layer and then use the Move Up or Move Down button to move the layer's position in the drawing queue.
