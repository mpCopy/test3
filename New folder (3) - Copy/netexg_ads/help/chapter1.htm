Introduction

PCB Importer is a program that extracts physically connected nets from ODB++ and IPC-2581 PCB formats by collecting together all touching or overlapping geometries on conductor layers. Drill data used to create vias is provides connection information from layer to layer. Component data is used to identify pins and reference designators. Net data is used to label groups of conductors.

When all of this information has been processed the program writes a file known as ADFI which can be imported intelligently by ADS. An ADS substrate definition is created based on the stackup information found in the ODB++ or the IPC2581 file.


Additional Features

Because printed circuit board layouts can be large and complex, the user can select only a portion of the layout to import into ADS. There are two separate selection modes:

By Window

In this mode the user selects (graphically) a window (or windows) to extract. All conductors are clipped at the window boundary.

By Net

In this mode the user selects one or more nets to extract. In addition to these nets, the user can select a proximity distance and all conductors that fall inside of the net's footprint are also selected. This is very useful when one has a long critical net that crosses the board.
